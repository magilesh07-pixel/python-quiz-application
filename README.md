# Quiz Application (Python)

A CLI-based Quiz Application built using Python.

## Features
- Multiple choice questions
- Score calculation
- Input validation
- Menu-driven interface

## Concepts Used
- Lists and dictionaries
- Loops and conditional statements
- User input handling

## How to Run
1. Download the file `quiz_application.py`
2. Run using:
   python quiz_application.py

This project was built as part of learning Python fundamentals.
